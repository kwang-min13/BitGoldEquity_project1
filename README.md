# 비트코인, 금, 그리고 주식 시장의 상관관계 분석
*(Correlation Analysis of Bitcoin, Gold, and Stock Markets)*

> **과제 프로젝트_2023**
> 
> "비트코인은 과연 주식 시장과 금 가격에 어떤 영향을 받을까?" 라는 호기심에서 출발한 프로젝트입니다. 실제 과거 데이터를 기반으로 각 자산 간의 상관관계를 분석하고, 선형 회귀 및 규제 모델을 통해 비트코인 가격의 설명력을 평가해 보았습니다.

---

## 프로젝트 개요 (Overview)
비트코인과 관련된 여러 기사나 정보들을 살펴보면, 비트코인이 **금의 가격 및 주식 시장과 상관관계가 있다**는 주장을 종종 접할 수 있습니다. 본 프로젝트는 이러한 가설을 검증하고자, 미국 주식 시장(나스닥 100, 다우존스, S&P 500) 및 금 가격 데이터를 바탕으로 비트코인 가격에 미치는 영향을 데이터 기반으로 확인하는 것을 목표로 합니다.

---

## 데이터 수집 및 전처리 (Data Preparation)

### 1. 데이터 수집
- **수집 기간**: 2018년 1월 1일 ~ 2023년 4월 30일 (일별 과거 데이터)
- **출처**: 
  - 주식 및 비트코인 (INVESTING.com) : 비트코인(Bitcoin), 나스닥 100(NDX), 다우존스(DJI), S&P 500
  - 금 (세계 금 협의회) : 달러 기준 금 과거 데이터

### 2. 데이터 전처리
- **날짜 기준 병합**: 금의 2018년 이전 데이터를 제외하고, 주식 시장의 영업일 기준으로 모든 자산의 종가 데이터를 `Merge`.
- **결측치 및 타입 정제**: 쉼표(`,`) 제거 및 `string` 변수를 `float` 형으로 변환. 날짜 데이터를 `datetime` 형식으로 일치화.
- **스케일링 및 변동성 조정**: 데이터의 스케일 차이를 맞추고 변동성을 최소화하기 위해 **로그 변환(Log Transformation)** 적용.

---

## 분석 및 모델링 (Analysis & Modeling)

### 1. 다중선형회귀 (Multiple Linear Regression - OLS)
- 각 독립변수 간의 상관관계를 확인한 후 선형 회귀(OLS) 모델을 학습했습니다. 
- 학습 결과, Train 데이터에 대해서는 높은 결정계수를 보였으나 Test 데이터에 대해서는 MSE(0.020)와 R²(0.201) 값 측면에서 유의미한 설명력을 가지기에는 부족함을 확인했습니다.

### 2. 규제 모델 (Ridge & ElasticNet)
과적합 방지와 더 나은 결과 도출을 위해 규제 강도(Alpha)를 조정하며 실험을 진행했습니다.
- **릿지(Ridge)**: $\alpha$ 값을 0.001에서 100까지 적용하며 성능 비교. $\alpha=100$ 일 때 (Train MSE: 0.171, Test MSE: 0.016 / Train R²: 0.749, Test R²: 0.371)을 기록.
- **엘라스틱넷(ElasticNet)**: L1, L2 규제 비율을 0, 0.2, 0.5, 0.7, 1 등으로 나누어 탐색.

## 한계점 및 배운 점 (Lessons Learned)
- **한계점**: 금융 시계열 데이터의 특성상 단순한 선형 회귀 모델만으로는 비트코인 가격을 유의미하게 예측하고 설명하는 데 한계가 있었습니다.
- **배운 점**: 
  1. 서로 다른 포맷을 가진 출처의 데이터를 가공하고, 날짜를 기준으로 병합(join)하는 **실전 데이터 핸들링 역량**을 키울 수 있었습니다.
  2. 단순 모델링을 넘어 MSE, R² 등 평가지표를 바탕으로 모델을 진단하고, 성능 개선을 위해 **규제 모델(Ridge, ElasticNet)**과 파라미터 튜닝을 주도적으로 시도해 보았습니다.
  3. 향후 시계열 데이터 전처리 역량을 키우고 예측에 특화된 모델(ARIMA, LSTM, Tree-based 머신러닝 모델)이나 파생 변수 생성을 도입한다면 모델 성능을 더 고도화할 수 있을 것이라 판단합니다.
